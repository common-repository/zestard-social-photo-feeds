Display customizable, and responsive social photo feeds from Instagram accounts

== Description ==

Display social photo feeds from instagram accounts

== Features ==

- User can use enable/disable option.

- User can set social photo feeds details from the backend.

- User just need to click Generate Instagram Access Token to generate access token.

- This plugin will generate access token automatically after login/authorization of user. 

- User can fully customize the style so it fits in your any page, post.

- Display instagram social photos feeds in sidebar by adding shortcode in widget.

- User can choose which layout(carousel/grid) you want to add in your social photo feeds.

- User can also style "Load More" button.


== Installation ==

1. Upload the `zestard-social-photo-feeds` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to "Settings/Social Photo Feeds" to configure the plugin.
4. Click on Generate Instagram Access Token and this plugin will generate access token for you.
5. After generating access token small popup will open and you just need to click on Continue button to proceed with plugin.
6. Click on "Save Changes" button.

== Display Your Feed ==

You can simply copy "[zestard-social-photo-feeds]" shortcode and paste in your post, page or any template file.

== Frequently Asked Questions ==
